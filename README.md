# 国学小书房 📖

幼小衔接国学启蒙学习游戏，面向 **4~8 岁儿童**。通过图形动画 + 互动练习结合的形式，帮助孩子建立汉字认知、古诗文语感、传统文化素养和语言表达能力。

## 技术栈

| 层次 | 技术 |
|------|------|
| 游戏引擎 | Phaser 4.2.0 (WebGL/Canvas) |
| 构建工具 | Vite 8.1 + TypeScript 6.0 |
| 部署产物 | 单文件 HTML（纯前端，`file://` 可直接运行） |

**架构特点：** 纯 Phaser 实现，无 React / Zustand / Tailwind 等 JS 框架依赖。所有 UI（菜单、关卡选择、游戏画面、暂停菜单、学习报告）均由 Phaser 场景原生渲染。

## 快速开始

### 开发模式（需要 Node.js）

```bash
npm install        # 安装依赖（仅 phaser + vite）
npm run dev        # 启动开发服务器 http://localhost:3000
```

### 生产构建

```bash
npm run build      # 构建单文件部署包
```

构建产物 `dist/index.html` 是一个**自包含的单文件**（约 1.5 MB），**双击即可在浏览器打开**，无需任何服务端支持。

## 场景流程

```
Boot → Preloader → MainMenu
                    ├── LevelSelectScene → ChineseGame → LevelComplete
                    └── ReportScene
```

| 场景 | 功能 |
|------|------|
| `Boot` | 初始化，立即跳转 Preloader |
| `Preloader` | 显示加载进度条（毛笔书写动画），加载资源后跳转主菜单 |
| `MainMenu` | 游戏标题（国学小书房）、统计卡片、功能说明、**开始学习**、**今日诵读**（随机古诗/三字经）、**成语接龙**快捷入口 |
| `LevelSelectScene` | 5 个分类 tab（水墨风格标签）、可滚动关卡卡片网格（卷轴样式）、底部统计栏 |
| `ChineseGame` | 核心答题场景，支持 15+ 种题型（详见下文） |
| `LevelComplete` | 成绩展示、评级动画（梅花/竹子等级）、墨迹晕染特效、重试 / 下一关 / 返回 |
| `ReportScene` | 学习总览：已学汉字量、古诗背诵数、成语掌握数、各模块进度条、学习建议、重置进度 |

## 核心功能

### 视觉风格 — 水墨国风

- **整体色调：** 宣纸米黄底色（#F5E6C8）、墨色文字、朱红点缀
- **字体：** 使用楷体/宋体风格，标题使用毛笔书写效果
- **背景装饰：** 水墨山水/竹梅/祥云等 SVG 元素作为背景层
- **过渡动画：** 墨迹晕染效果（页面切换）、卷轴展开（关卡选择）
- **正确反馈：** 水墨散开 + 梅花飘落
- **错误反馈：** 墨滴溅落效果
- **过关特效：** 毛笔书写"太棒了！"动画 + 烟花粒子

### 音效系统

- 用 Web Audio API 直接合成音效（无需音频文件）
- **正确音效：** 古筝/琵琶短音（上升音调）
- **错误音效：** 编钟低音（下滑音）
- **连击 3/5/10 特殊音效：** 古琴旋律
- **过关庆祝音效：** 笛子吹奏
- **背景音：** 古典轻音乐（可选开关）

### 进度系统

- 自动保存至 `localStorage`（key: `chinese-study-levels`）
- 保存每关最高分、最高正确率、梅花评级、最快完成时间
- **梅花评级：** 1 朵（完成）、2 朵（正确率 ≥80%）、3 朵（≥95%）
- 关卡解锁：同类顺序解锁，跨类型需要累计分和平均正确率达标

### 答题模式

- **诵读模式：** 逐句显示经典内容，跟读后选字/填空，不限时
- **挑战模式：** 每关限时（20s / 15s / 10s），时间到扣生命
- **自由模式：** 全部关卡解锁后自由选择已过关卡复习

### 学习报告

- **汉字银行：** 累计学会的汉字数量（按关卡统计）
- **古诗背诵：** 已背古诗数量 + 列表
- **成语库：** 已掌握的成语数量 + 列表
- **模块进度：** 各模块完成率 + 梅花数
- **学习建议：** 根据薄弱环节推荐练习关卡
- **重置所有进度**

## 关键设计决策

### 1. 为什么不用 React？

同 `study_math`，Phaser 的 Canvas 坐标与 React 组件布局之间存在难以调和的点击偏移问题，且 `file://` 协议下 ES Module 加载受限。

### 2. 国风字体处理

```
系统回退链：'KaiTi', 'STKaiti', 'SimSun', 'STSong', serif
```

不内嵌字体文件（太大），利用系统字体。Windows/macOS 均预装楷体/宋体。

对于标题使用的毛笔书法字体（如"国学小书房"），使用 **Phaser 文字纹理** 预先渲染为图片纹理，避免调用外部字体。

### 3. 拼音标注方案

```
使用 Phaser 的 Text 对象分层渲染：
┌──────────────────┐
│  rén zhī chū     │ ← 拼音行（小字号，gray，y 偏移）
│  人之初           │ ← 汉字行（大字号，black）
└──────────────────┘
```

拼音动态生成（用 `pinyin-pro` 库或内置拼音映射表），标注在汉字上方。

### 4. 画布缩放策略

```
Phaser Scale: FIT, NO_CENTER
CSS: position: absolute + inset: 0 + margin: auto
```

### 5. 击中区域（Hit Area）修正

同 `study_math`，所有交互式 Container 统一使用 `Rectangle(0, 0, w, h)`。

### 6. 单文件构建流程

同 `study_math`：Vite build → inline-assets.cjs 内联 JS/CSS → 单文件 HTML。

### 部署方式

- ✅ **直接使用：** 双击 `dist/index.html`，在浏览器打开

### 关卡配置

编辑 `src/game/utils/LevelSystem.ts` 中的 `LEVEL_CONFIGS` 数组：

```typescript
{
  id: 1,
  name: '三字经·人之初',
  description: '人之初，性本善',
  module: 'classics',       // 模块: classics / poetry / idiom / char / culture
  type: 'fill-blank',       // 题型类型
  difficulty: 1,            // 难度 1~5
  requiredScore: 0,         // 解锁所需总分
  requiredAccuracy: 0,      // 解锁所需平均正确率%
  problemsCount: 10,        // 每轮题数
  sourceId: 'sanzi-01',     // 关联内容 ID
}
```

可用 `type`：`fill-blank` / `sequence` / `match-pair` / `image-select` / `classify` / `drag-match` / `find-diff` / `chain` / `sort` / `story-sort` / `choice`

### 题目池配置

固定题在 `src/game/scenes/ChineseGame.ts` 对应 `showXXXProblem()` 方法中。直接在数组中增删题目即可。

具体题源数据（古诗/成语/汉字库）统一放在 `src/game/data/` 目录下，以 JSON / TypeScript 常量形式管理：

```
src/game/data/
├── classics.ts       # 蒙学经典数据
├── poems.ts          # 古诗数据
├── idioms.ts         # 成语数据
├── characters.ts     # 汉字数据
└── culture.ts        # 传统文化数据
```

### Vite 配置

关键项（`vite.config.ts`）：
- `base: './'` — 确保构建后资源路径为相对路径
- `assetsInlineLimit: 100000000` — 大资源也内联为 base64
- `codeSplitting: false` — 单文件模式，不拆分包

## 已知问题 / 注意事项

- **System font 回退差异：** 不同操作系统预装字体不同，楷体在 macOS 上为 `STKaiti`，在 Linux 上需安装。标注字体回退链 `'KaiTi', 'STKaiti', 'SimSun', 'serif'`。
- **拼音多音字处理：** 古诗文中多音字较多（如"见"在"风吹草低见牛羊"中读 xiàn），内置拼音映射表需人工校验，不能完全依赖自动标注。
- **象形图绘制：** 使用 Phaser 图形 API（Graphics）绘制简化象形符号，不依赖外部图片资源。每个象形字约 20~40 行绘制代码。
- **长文本滚动：** 古诗全文/成语故事展示时，如果文字超出画布，使用 Phaser 的 ScrollFactor 或手动实现滚动画板。
- **点击偏移：** 同 `study_math`，如遇点击偏移，调用 `game.scale.refresh()` 修正。
